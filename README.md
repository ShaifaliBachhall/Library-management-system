# 📚 Library Management System

## 📌 Overview
A console-based Library Management System built using Java and JDBC that helps manage books, users, and issue/return operations efficiently with database integration.

## 🚀 Features
- Add and manage books
- Issue books
- Track availability of books
- Maintain user records
- Database connectivity using JDBC

## 🛠️ Tech Stack
- Java
- JDBC
- MySQL
- IntelliJ IDEA

## 📈 Future Improvements
- GUI (JavaFX/Swing)
- Login system
- Online reservation system
